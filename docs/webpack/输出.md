
# 输出（output）

输出 output 选项可以控制webpack 如何向硬盘写入编译文件。

> 注意：即使可以存在多个入口文件，但只能指定一个输出配置。

## 用法

在webpack 配置中，output属性的最低要求是，将它的值设置为一个对象，包括以下两点：

+ filename 用户输出文件的文件名
+ 目标输出目录 path 的绝对路径

webpack.config.js
```js
const config = {
    output:{
        filename:'bundle.js',
        path:'/'
    }
}
```
此配置讲一个单独的 bundle.js 文件输出到 / 目录中
