
# 入口起点 （entry points）

## 单个入口语法

<!-- 用法：entry:string|Array<string> -->

webpack.config.js

```js
const config ={
    entry:'./path/to/my/entry/file.js'
};
module.exports=config;
```

是下面简写：

```js
const config ={
    entry:{
        main: './path/to/my/entry/file.js'
    }
};
module.exports=config;
```
